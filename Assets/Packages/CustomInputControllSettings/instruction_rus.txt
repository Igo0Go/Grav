Этот пакет используется для внедрения смены управления в игру.  Легче всего понять, как это работает с помощью вложенной сцены с примером. Там вы управляете игроком от первого лица, а при нажатии ESCAPE можете сменить управление. 

В префабах есть элементы интерфейса: 
- панель для настроек управления 
- элемент этой панели для настройки конкретной кнопки 

Для того, чтоб всё работало нужно создать файл по типу TestInputPack. Это пакет настроек. По инспектору там всё должно быть понятно. 
Можно создать кнопки: 
- строка, по которой будем искать в коде 
- сама кнопка клавиатуры или мыши 
- строка, которая будет использоваться для обозначения того, за что отвечает кнопка, в интерфейсе уже в игре. 
Также из кнопок можно составлять оси. Они используются когда нужно плавно менять значения в пределах от -1 до 1. 
Создать такой пакет можно нажатием правой кнопки мыши, затем Create/Config/InputKit. Панель при старте сама будет погружать все кнопки из пакета. Главное верно указать все ссылки (смотри пример).
